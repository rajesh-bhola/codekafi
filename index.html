About Apache Kafka

Apache Kafka is an open-source stream-processing software platform which is used to handle the real-time data storage.


About KsqlDB

The database purpose-built for stream processing applications.


About Kafka Streams

Kafka Streams is a light-weight in-built client library which is used for building different applications and microservices.


About Kafka Connect

It is an open-source component and framework to get Kafka connected with the external systems.
