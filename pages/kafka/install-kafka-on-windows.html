Welcome CrazyDev, This article show you the steps to set up and run open source apache kafka on windows.


Step 1:

To install Apache Kafka, the JDK 8 should be installed on the system.
If JDK 8 not installed on your system then go ahead and download and install it.


Step 2:

Download Apache Kafka on the system. Click below link to download kafka from official website.

https://kafka.apache.org/downloads

Then download from Binary Downloads section. Chose any version under Binary Downloads. Now you can redirected to another page, click on the first link to download '.tgz' file. And save the file on your desktop.


Step 3:

Go to the folder where you have downloaded the kafka and extract the file using linux terminal or 7zip or WinRar software.


Step 4:

After extraction, go to the kafka folder then config folder and double click on server.properties file. We need to change some broker configuration here.

server.properties
Uncomment and changed the host name to localhost for the following:

listeners=PLAINTEXT://localhost:9092

advertised.listeners=PLAINTEXT://localhost:9092

log.dirs=D:/RAJESH/kafka/kafka-logs


Now, you are ready with the configurations. It's time to run the zookeeper and broker.


Run Zookeeper

Go to kafka folder inside and open cmd prompt and type the following:

D:\RAJESH\kafka > .\bin\windows\zookeeper-server-start.bat  .\config\zookeeper.properties



Run Broker

Go to kafka folder inside and open cmd prompt and type the following:

D:\RAJESH\kafka > .\bin\windows\kafka-server-start.bat  .\config\server.properties
