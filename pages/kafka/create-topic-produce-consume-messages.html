Welcome CrazyDev, This article show you how we can create topics, produce some messages on the console and consume it from consumer console.


In the previous blog we learnt how to install Kafka on windows. If you are not aware of this then please visit the below link:

How to install Kafka on windows


Without any further discussion let's deep dive into creating a topic.


Create a new topic

At the time of creation we should mention the partitions and replication factors. I believe, you already have running the zookeeper and broker in your system. If you are not, then run it.


Open another cmd prompt and type the following command to create a new topic.

.\bin\windows\kafka-topics.bat  ‚Äìcreate  ‚Äìzookeeper  localhost:2181  ‚Äìreplication-factor  1  ‚Äìpartitions  1  ‚Äìtopic  demo-topic


Now, the topic named "demo-topic" got created. To check the list of topics in kafka type the following command in the same cmd prompt.


List of topics

.\bin\windows\kafka-topics.bat  --list  --zookeeper  localhost:2181



Now, we can describe the topic by using the following command.

Describe a topic

.\bin\windows\kafka-topics.bat  --describe  --zookeeper  localhost:2181  --topic  demo-topic



Run producer console

Open a new cmd prompt and type the following:

.\bin\windows\kafka-console-producer.bat  ‚Äìbroker-list  localhost:9092  ‚Äìtopic  demo-topic


Then type some messages on the producer console which can be shown in the consumer console.

> Hello world!

> Welcome to apache kafka

> This messages from the producer console



Run consumer console

Now to see the messages produced by producer console, open a new cmd prompt and type the following:

.\bin\windows\kafka-console-consumer.bat  ‚Äìbootstrap-server  localhost:9092  ‚Äìtopic  demo-topic  ‚Äìfrom-beginning
